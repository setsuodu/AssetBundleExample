# AssetBundleExample

## Introduction

a template to show how to build and load assetbundle.

## WorkFlow

- create ABs;
- zip;
- upload;
- download;
- unzip;
- LoadAsset;

## API, platform path and permission

- Application.persistentDataPath

- Application.temporaryCachePath

- Application.streamingAssetsPath

- Application.dataPath
